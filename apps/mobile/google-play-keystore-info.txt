# ==========================================
# COMO GERAR O APLICATIVO PARA A PLAY STORE
# ==========================================
Como você está usando o Expo, a melhor forma de gerar o arquivo ".aab" (Android App Bundle) exigido pela Google Play é utilizando o EAS Build.

1. Instale o EAS CLI (se ainda não tiver):
   npm install -g eas-cli

2. Faça login na sua conta do Expo:
   eas login

3. Inicie a configuração do projeto no EAS:
   eas build:configure

4. Gere o build final para a loja:
   eas build --platform android --profile production

DICA: Quando o EAS perguntar se você deseja gerar uma nova chave de assinatura automaticamente, você pode dizer "Yes" (recomendado, pois a Expo gerencia pra você com máxima segurança) OU pode selecionar para fazer o upload do arquivo "release.keystore" gerado aqui, informando o Alias e a Senha acima.

# ==========================================
# PUBLICANDO NA GOOGLE PLAY CONSOLE
# ==========================================
1. Crie uma conta de desenvolvedor na Google Play Console (custa $25 uma única vez).
2. Clique em "Criar App", preencha o nome ("Fiado") e os detalhes.
3. Vá em "Testes" ou "Produção", crie uma nova versão.
4. Faça o upload do arquivo ".aab" gerado pelo EAS Build.
5. Preencha a classificação indicativa, imagens, ícone e descrição.
6. Envie para revisão da Google!
